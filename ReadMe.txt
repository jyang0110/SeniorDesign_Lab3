For running lab3:

1. Make sure the I and Q channel audio files are named as 'distroted_I.wav' and 'distroted_Q.wav'

2. Make sure the 'lab3.m' file is in the same folder as these audio files

3. Make sure the 'wavchunksizefix.m' is also in the same folder

4. Open the 'lab3.m' and click to run the code, the undistorted audio will be output to the same folder and named as 'undistorted_output.wav'
